📌 *Sobre o Projeto*

Este projeto foi desenvolvido como parte da disciplina de Programação Orientada a Objetos (POO).
O sistema implementa uma biblioteca, permitindo realizar operações básicas como cadastro de usuarios, cadastro, remoção, 
emprestimo, devolução e listagem de livros.

 *Objetivos*

Aplicar na prática os principais conceitos de POO, como:

1.Encapsulamentp
2.Herança
3.Polimorfismo
4.Abstração

🖥️*OPERAÇÃO VIA TERMINAL*

O sistema aparece assim:

1 - Cadastrar livro
2 - Cadastrar usuário
3 - Emprestar livro
4 - Devolver livro
5 - Listar livros disponíveis
6 - Listar empréstimos
0 - Sair
